Практическая работа №2

Создание базы данных, таблиц и добавление данных

Цель работы:
  • научиться создавать базы данных, таблицы и заполнять их данными.

Ход работы

SQLite Studio представляет собой программу, которая предназначена для создания и редактирования баз данных. При помощи данного ПО можно легко создавать и редактировать таблицы, писать запросы на языке SQL.

Создание базы данных

  1. Для создания новой БД SQLite войдите в меню «База данных — Добавить базу данных».
  2. Появится диалог подключения БД, в котором следует ввести имя файла и имя базы данных. Имя БД — это просто имя для отображения в списке. Напишите то, какое вам удобно.

Создание таблицы

  1. Для создания таблицы войдите в меню «Структура — Создать таблицу».
  2. Появится окно структуры. Введите имя таблицы, например, «Организации».

Добавление полей и данных

  1. Теперь нужно добавить поля. Нажмите клавишу Insert. Откроется окно, позволяющее задать имя столбца, тип данных, размер и поставить ограничения.
  2. Добавьте три поля: «Название, Адрес, Телефон» и нажмите зеленую галочку.
  3. Появится окно «Запросы, которые будут выполнены». После нажатия кнопки «ОК» таблица будет создана.
  4. Для ввода информации в созданную таблицу нажмите на вкладку данные, далее на клавишу Insert и введите необходимые данные, после чего нажмите на зеленую галочку
  5. Заполните таблицу «Организации» 5 строками данных.
